# 🔤 Token Sampling Techniques in Python

This project demonstrates fundamental token sampling techniques used in Large Language Models (LLMs). It includes examples of Next Token Prediction, Temperature Scaling, Top-K Sampling, and Top-P (Nucleus) Sampling using Python and NumPy.

## 🚀 Features

- Next Token Prediction
- Temperature Scaling (basic implementation)
- Top-K Sampling
- Top-P (Nucleus) Sampling
- Beginner-friendly Python implementation
- Lightweight and easy to understand

## 🛠️ Technologies Used

- Python
- NumPy
